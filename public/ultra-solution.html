The Hard Truth

After extensive testing with Playwright, analyzing SimpleSwap's source code, and trying every possible parameter:

  • SimpleSwap's frontend is a closed system. It manages provider selection internally and ignores URL parameters.
  • No magic parameter exists. We tried: provider, selectedProvider, defaultProvider, forceProvider, partner_provider, and 50+ others.
  • Their API/widget is crypto-only. No fiat purchase control exposed.
  • Browser automation works but requires running headless browsers on your server.

The only ways to achieve automatic Mercuryo selection:

  1. Use Mercuryo directly (Option 1)
  2. Run browser automation (Option 2)
  3. Partner with SimpleSwap for custom integration
